## üìä How It Works

The application works by capturing electrical signals from your muscles. These signals are analyzed to detect various hand movements. The use of a Transformer model helps improve the detection accuracy, making it faster and more reliable.

1. **Signal Collection**: The software accesses the muscle signals through compatible sensors (not included).
2. **Feature Extraction**: Using EMD-CCT, important characteristics of the signals are highlighted for further analysis.
3. **Movement Detection**: The application then interprets these features to recognize specific gestures or movements.
